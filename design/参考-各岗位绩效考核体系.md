# 参考：各岗位绩效考核体系

> 来源：`各岗位薪资绩效/` 全文件全 sheet
> 用途：校准 TDL 的周期任务模板、关键动作标签、评分→催收链路

## 一、统一的评分机制

所有岗位共用同一套 5 分制：

| 评分区间 | 系数 | 含义 |
|---------|------|------|
| ≥4.6 | 1.5 | 优秀 |
| 3.6-4.5 | 1.2 | 良好 |
| 3.0-3.5 | 1.0 | 合格 |
| 2.0-2.9 | 0.7 | 需改进 |
| <2 | 0 | 不合格 |

评分人 = 直接上级。评分需要证据支撑（数据、截图、周报、复盘文档）。

## 二、各岗位 KPI 维度（= TDL 关键动作）

### 人力
- **专员 (Judy/Vivi)**：每周视频收取量（≥10）、周到面量（≥4人）、行政专项事务（社保/工商/宿舍管理，0差错）→ 评分人：张皓
- **人力经理 (张皓)**：招聘人头提成 + 月度述职评分 + 季度绩效 → 评分人：李珍/上级

### 市场
- **新媒体运营（线上）**：底薪 3000 + 过程奖金 1600 + 结果提成（到访 5元/人、成单 150元/人）+ 销售完标奖 300元
- **市场专员（线下）**：固定 3000 + 绩效基数 2000（过程行为金 + 结果提成 + 销售完标奖）+ 直播渠道跳板式单价
- **市场经理 (时颖)**：管控市场成本（<600元/人）、整合市场资源（线上线下渠道拓展）、巡检市场专员（周报体现巡检+帮带+效果回收）→ 评分人：李珍

### 教务和班主任
- **班主任 (T1-T4)**：班级数据管理（0.2权重：按时提交、完整学员档案）、升班管理（0.3权重：34周前费用统计、36周前续费电话、跟盯数据）、家长沟通维护（0.3权重：公开课邀请函回收、沟通记录、问题同步）→ 评分人：赵晓华
- **教务经理 (赵晓华)**：基本 6000 + 带班费 25元/班 + 季度绩效 4500

### 教学
- **教师 T2**：月度绩效基数 300。评分维度：教学项目（黄金五分钟 PPT、学习效果提升方案、绘本阅读项目）→ 评分人：教学主管
- **教师 T3**：月度绩效基数 500。每月选 2 个项目评分（过课/教研培训/教学项目），每个月项目权重可调
- **教学 AD（年级主管）**：季度考核=升班奖金+评分奖金。季度评分维度：满班率（0.2，目标≥13.5）、转正中教时段比（0.15，目标≥7）、ref 成单占比（0.25，目标≥6%）
- **培训师**：月度述职评分 + 季度绩效（基数 3000）+ 年度达标奖金（升班率≥85%）

### 销售
- **课程顾问**：跳人头提成（1-3单100元/单 → 10单300元/单）
- **销售经理 (时颖)**：ccm 电话时长（每周≥2小时，0.25权重）、电话录音抽查（每周≥6条，0.25权重）、盘单（demo 前盘单+demo 后复盘，0.3权重）、ccm 成单（月度≥4单，0.2权重）。季度考核：满班开班率（目标开班平均 14 人）

## 三、对 TDL 系统的直接影响

### 1. 每月固定的"评分截止日"是一个全公司级别的 TDL

每个月，上述所有评分人（张皓、时颖、赵晓华、曹艳明、邢杨、张蕾）都要完成下属评分。这些评分必须交到薪资系统（张皓/许梦冉汇总基础薪资表）。

这意味着 TDL 系统里有一条跨部门的 `@recurring` 任务链：
```
月初设定目标 → 月度收集证据 → 月底下属述职 → 上级评分 → 提交薪资系统（12号截止）
```

### 2. 评分本身就是管理闭环的"回收"环节

Codex 的方案提到评价管理者的三条线（发言质量/行动证据/结果兑现）。这些已有的绩效考核维度可以直接作为第三条线的输入——不需要 AI 重新发明一套评分标准，AI 做的是"帮你把评分过程自动化、证据收集自动化、趋势可视化"。

### 3. 评分趋势数据直接喂给"沉淀进化"

连续 3-6 个月的评分趋势、同一个教师从 T2 到 T3 的评分变化、离职教师最后 3 个月的评分下滑——都是沉淀进化的天然数据源。
