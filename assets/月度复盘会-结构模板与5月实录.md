# 月度复盘会：结构模板与 5 月实录

> 来源：2026年5月9日励步月度会（原文 + 6份述职材料）
> 用途：月会办会参考、AI 月会闭环原型设计

## 一、月会固定结构（从5月会提炼）

### 常规议程板块

1. **公司级信息同步**（李珍）：AI 进展、系统建设、战略方向
2. **前端运营复盘**（时颖）：销售数据 + 市场数据 + 渠道分析 + 问题诊断 + 下月策略
3. **教学部门复盘**（各教学主管）：升班数据、教师状态、续费进度、课质效果
4. **职能支撑复盘**（赵晓华/张皓/韩雪敏）：教务看板、人力招聘与培训、新师成长
5. **专项讨论**：新师培训方案、校区运营策略、跨部门协同问题
6. **管理方法论**（李珍）：基于现场发现的管理短板，现场辅导

### 会议输出物

- 决议清单（责任人 + 截止时间）
- 经营地图（各部门目标/进展/风险）
- 管理者成长诊断
- 下次会议追踪点

## 三、本次月会发现的管理问题

1. **目标清晰但落地路径缺失**：管理者知道要什么，但"怎么让他会"的标准动作没设计
2. **执行力看板推进困难**：70+ 教师每周评分，教务经理压力大，看板规则争议多
3. **新师安排半数待定**：19 名新师中过半"后续安排待定"，教学主管和教务经理之间信息不对称
4. **教师稳定性需逐个沟通**：多位教师有离职意向（April 6月离职、Rita 5月离职、Carina 6月离职），需逐月跟踪
5. **市场线上线下打通困难**：线上流量难分配到具体校区，线下活动效果评估标准不统一
