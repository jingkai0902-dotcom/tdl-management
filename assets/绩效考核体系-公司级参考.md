# 绩效考核体系：公司级参考

> 来源：各岗位薪资绩效文件（全6文件全sheet）
> 用途：绩效管理、TDL 评分标准设计、管理者考核

## 一、通用评分标准

全公司共用 5 分制：

| 评分 | 系数 | 含义 |
|------|------|------|
| ≥4.6 | 1.5 | 优秀——超预期达成，形成可复制方法论 |
| 3.6-4.5 | 1.2 | 良好——达成目标，有复盘总结 |
| 3.0-3.5 | 1.0 | 合格——按计划完成 |
| 2.0-2.9 | 0.7 | 需改进——经过大量帮扶才达成 |
| <2.0 | 0 | 不合格——未达成目标 |

评分人 = 直接上级。评分需证据支撑（数据、截图、周报、复盘文档）。

## 四、管理洞察（从绩效体系可推）

1. **单人评分负荷**：时颖同时承担销售经理自评 + 市场经理 + 运营经理三个角色的评分责任
2. **评分覆盖盲区**：苗雅楠直管的销售/市场团队（均入职未满一年）绩效评分体系尚不完整
3. **季度与月度脱节风险**：月度评分和季度评分由不同人执行（如教学：月=教学主管、季=张蕾），需信息同步
4. **培训师角色模糊**：韩雪敏的绩效既有教学指标又有培训指标，归属教学培训标签
