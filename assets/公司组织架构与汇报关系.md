## 标准校区结构（理论模型）

```
                    校长
                     │
     ┌───────────┬───┴───┬───────────┐
     │           │       │           │
   市场         销售     教学        职能部门
                                     │
                          ┌────┬────┬┴───┬────┐
                          │    │    │    │    │
                         教务 班主任 行政  人力  ...
```

铁三角（市场+销售+教学）是校区运营核心。各团队负责人职级从主管到经理。
